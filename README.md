# 🌐 Geek-Print33

---

## 🛠️ Стек технологий

- **Backend:** PHP / Laravel
- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **База данных:** MySQL
- **Другое:** AdminLTE, AJAX, REST

---

## ✨ Функционал

- 🔐 Авторизация админа
- 📦 Каталог товаров
- 🔍 Поиск и фильтрация
- 🧩 CRUD (товары, категории, теги)
- 📊 Админ-панель
- 📱 Адаптивная вёрстка
- 🤖 Телеграм бот
